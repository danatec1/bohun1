🎨 배경색 강제 테스트

이 페이지는 모든 스타일에 !important를 사용합니다.

만약 이 텍스트가 검은색으로 보이지 않거나
배경이 흰색이 아니라면,
시스템이나 브라우저의 다크모드/고대비 모드가 활성화되어 있을 수 있습니다.

#FFFFFF
순수 흰색
#FAFAFA
밝은 회색
#F5F5F5
연한 회색

⚠️ 배경이 흰색이 아닌 경우

  • Windows 설정 → 접근성 → 고대비 모드 확인
  • Windows 설정 → 개인 설정 → 색 → 다크/라이트 모드 확인
  • 브라우저 확장 프로그램 (다크 리더 등) 비활성화
  • 브라우저 설정 → 모양 → 테마 확인
  • VS Code Simple Browser의 테마 설정 확인
